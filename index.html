My Projects

Study Bot

A Full Stack Web app built using Javascript and React for the Frontend, Python and Flask for the backend and MYSQL for the database.

The App is a study planner which allows users to add classes and events and view these in a calendar. It also uses Machine Learing to summarise Lecture Notes and create questions to quiz users to optimise their learning.

Traffic Simulator

This project is implemented using a MVC design pattern in C++ using Linked Lists. This is a time-discrete and space-discrete model. During each tick a car can move forward one step known as car following or change lane to a adjacent lane known as lane changing. These movements are allowed as long as there aren't any conflicts.
